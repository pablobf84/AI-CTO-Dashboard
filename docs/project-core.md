# AI CTO Dashboard — Project Core

## Qué es
AI CTO Dashboard es un SaaS GitHub-first que convierte una idea de SaaS en:
- PRD
- arquitectura técnica
- backlog por fases
- prompt packs para agentes
- memoria de proyecto
- aprobaciones por etapa

## Qué no es
No es otro builder genérico de apps con IA.
No compite por “hazme una app”.
Compite por:
- claridad
- control
- memoria
- coordinación entre agentes
- trazabilidad de decisiones

## Cliente ideal inicial
- founders no técnicos o semi-técnicos
- agencias IA / automatización
- estudios pequeños de producto

## Problema que resuelve
Los usuarios tienen acceso a varias IAs, pero no tienen un sistema que:
- convierta idea en especificación útil
- organice el trabajo
- mantenga memoria
- controle aprobaciones
- reduzca caos entre herramientas

## Promesa
“Convierte una idea en un sistema de ejecución ordenado para construir tu SaaS con varias IAs sin perder control.”

## MVP
Incluye:
- autenticación
- organizaciones
- dashboard
- creación de proyecto
- intake de idea
- generación de PRD
- generación de arquitectura
- backlog por fases
- prompt packs para agentes
- memoria de proyecto
- aprobaciones por etapa

Excluye:
- ejecución automática profunda sobre todas las APIs de terceros
- deploy automático universal
- editor visual complejo
- colaboración avanzada
- billing completo en la primera semana
